# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Event

- `GET /api/event`
- `POST /api/event`
- `GET /api/event/:id`
- `PATCH /api/event/:id`
- `DELETE /api/event/:id`
- `SEARCH /api/event/search?query=?`

### Group

- `GET /api/group`
- `POST /api/group`
- `GET /api/group/:id`
- `DELETE /api/group/:id`
